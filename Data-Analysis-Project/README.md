# Data Analysis Project Using Python

## Overview

This project demonstrates data analysis using Python and Pandas.
The program loads a CSV dataset, cleans the data, performs filtering and grouping operations, generates summary statistics, and visualizes results using Matplotlib.

---

## Features

- Load CSV using Pandas
- Handle missing values
- Remove duplicate records
- Generate summary statistics
- Filter records based on conditions
- Group and aggregate data
- Generate meaningful insights
- Save cleaned dataset
- Create visualizations

---

## Technologies Used

- Python
- Pandas
- Matplotlib
